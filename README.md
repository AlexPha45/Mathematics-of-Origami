# Origami 101: Mathematics of Origami
**University of Toronto | Winter 2025**

This repository contains study notes and mathematical proofs exploring the intersection of geometry and origami.

## 📌 Core Concepts

### Flat-Foldability
To determine if a crease pattern can be folded flat, we analyze local and global restrictions.
*   **Maekawa's Theorem:** $|M - V| = 2$
*   **Kawasaki's Theorem:** $\sum \theta_{2i-1} = \sum \theta_{2i} = 180^\circ$

### Differential Geometry
Applications of curvature and manifolds to paper surfaces:
*   **Gaussian Curvature:** A local isometric invariant (Theorema Egregium).
*   **Gauss-Bonnet Theorem:** 
    $$\int_{S} K dA = 2\pi \chi(S)$$
    where $\chi(S) = V - E + F$.
